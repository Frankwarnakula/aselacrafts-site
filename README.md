# AselaCrafts Personal Branding Website

A modern React + Vite + TypeScript portfolio site for AselaCrafts and artist Asela Warnakula.

## Features

- Elegant premium hero section
- Artwork gallery using public YouTube thumbnail images
- YouTube and Facebook embed sections
- Social links for YouTube, Facebook, TikTok and Instagram
- Responsive mobile-first layout
- Smooth animations with Framer Motion
- Ready for deployment to Vercel, Netlify, Cloudflare Pages, or GitHub Pages

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Update social links

The YouTube and Facebook URLs are already included:

- YouTube: https://www.youtube.com/@aselacrafts
- Facebook: https://www.facebook.com/AselaCraft

Replace the placeholder TikTok and Instagram `href` values in `src/main.tsx` once the exact profile URLs are available.

## Recommended deployment

Vercel is the simplest option:

1. Push this folder to GitHub.
2. Import the GitHub repository in Vercel.
3. Use the default Vite settings.
4. Add a custom domain such as `aselacrafts.com`.
